一个基于WebUSB的ADB工具，用于在浏览器中管理Android设备，支持应用安装、系统工具等功能。

## 功能特性

### 核心功能
1. **设备管理**
   - 有线USB连接（通过WebUSB）
   - 设备状态显示
   - 无线ADB端口开启

2. **一键安装**
   - 虚拟返回键
   - 一键清理（支持定制版和通用版）
   - 沙发管家HD
   - 权限狗
   - 无障碍管理器
   - 返回菜单键
   - 氢桌面
   - 侧边栏

3. **应用管理**
   - 查看已安装应用列表（第三方应用）
   - 启动应用
   - 停止应用
   - 卸载应用
   - 安装本地APK文件

4. **快捷启动**
   - 启动应用管家
   - 启动氢桌面
   - 启动设置
   - 启动侧边栏

5. **系统工具**
   - 开启无线ADB端口
   - 解除网络防火墙
   - 解除安装限制

6. **命令执行**
   - 手动执行ADB shell命令
   - 命令输出日志显示

## 使用说明

### 环境要求
- **浏览器**：Microsoft Edge 或 Google Chrome（支持WebUSB）
- **设备**：Android设备
- **连接**：专用OTG线或数据线

### 基本操作
1. 使用Edge/Chrome浏览器访问工具页面
2. 点击"有线连接"按钮
3. 在设备上允许USB调试
4. 选择需要的功能按钮执行操作

### 连接步骤
1. 点击"有线连接"按钮
2. 浏览器弹出设备选择窗口
3. 选择您的Android设备
4. 在设备上点击"允许"
5. 连接成功后即可使用各项功能

### 常见问题
1. **连接失败**：尝试拔插数据线重新连接，或运行 `adb kill-server`
2. **浏览器不支持**：使用Edge或Chrome浏览器
3. **权限问题**：确保设备已开启USB调试模式
4. **root权限**：部分系统工具需要root权限

## 注意事项

1. 使用OTG线或数据线连接设备
2. 首次连接设备时需要在设备上允许USB调试
3. 部分功能需要设备具有root权限
4. 使用过程中请保持设备连接稳定
