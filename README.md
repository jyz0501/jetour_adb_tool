# 捷途大伦哥的应用管家

**在线访问**: https://jyz0501.github.io/jetour_adb_tool/

> ⚠️ **免责声明**：本工具仅供个人学习测试使用，请勿用于商业用途。使用本工具造成的任何后果由使用者自行承担。

一个基于WebUSB的ADB工具，用于在浏览器中管理Android设备，支持应用安装、系统工具等功能。

## 功能特性

### 安装方式
- **手机App方式**：直接下载手机助手App
- **WebUSB方式**：通过浏览器连接设备（需要Edge浏览器）

### 核心功能
1. **设备管理**
   - 设备连接/断开
   - 设备状态显示
   - 无线ADB开启

2. **应用安装**
   - 虚拟返回键
   - 一键清理（支持定制版和通用版）
   - 沙发管家HD
   - 大伦应用管家
   - 权限狗
   - 无障碍管理器
   - 返回菜单键
   - 氢桌面

3. **系统工具**
   - 激活大伦车机助手
   - 解除网络限制
   - 解除安装限制

4. **应用管理**
   - 查看已安装应用列表
   - 启动/停止/卸载应用
   - 安装本地APK文件

5. **命令执行**
   - 手动执行ADB shell命令
   - 命令输出日志显示

## 使用说明

### 环境要求
- **浏览器**：Microsoft Edge
- **设备**：Android设备
- **连接**：专用OTG线

### 基本操作
1. 使用Edge浏览器访问工具页面
2. 点击"连接设备"按钮
3. 在设备上允许ADB调试
4. 选择需要的功能按钮执行操作

### 常见问题
1. **连接失败**：尝试拔插数据线重新连接
2. **浏览器不支持**：使用Edge浏览器
3. **权限问题**：确保设备已开启USB调试模式
4. **root权限**：部分系统工具需要root权限

## 注意事项

1. 使用专用OTG线连接设备，普通数据线可能无法使用
2. 首次连接设备时需要在设备上允许ADB调试
3. 部分功能需要设备具有root权限
4. 使用过程中请保持设备连接稳定
